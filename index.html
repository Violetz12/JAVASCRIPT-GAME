Escape the cat!

Welcome to our interactive experience built with the magic of P5.JS! This is no ordinary game — it’s an escape room challenge where you and a friend must work together to outsmart a giant cat.

One of you plays a small mouse, the other a flying bird. Solve puzzles, activate buttons, and help each other escape before the cat catches you (jumpscare alert!). 🐭🐦🐱

Note: Best played on desktop or laptop devices. If the game doesn’t load properly, try resizing your browser window.

How to Play

This is a 2-player escape game — you’ll need teamwork and quick thinking to beat the clock!

  • Player 1: Use Arrow Keys
  • Player 2: Use W A S D
  • Each character has unique abilities:
    • One can fly 🕊️
    • One can slip through tight spaces 🐭
  • Work together to solve puzzles and find the way out!
  • Oh, and remember to finish before the time end... or... you’ll see 😼
